# smartcontract-sample

https://github.com/openethereum/openethereum

## MetaMaskのインストール
https://chrome.google.com/webstore/detail/metamask/nkbihfbeogaeaoehlefnkodbefgpgknn

```
npm i -g truffle
```

## コンパイル
```
truffle compile
```

## マイグレーション
```
truffle migrate --network development
```

## クライアント
```
cd clients
yarn
yarn start
```
